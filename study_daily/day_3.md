```
1. 数据可视化的后半部分，频率表/直方图/柱状图的各种用处；
2. 做了数据可视化相关的习题；
3. google spreadsheet；
4. 集中趋势：
  1. 众数：不一定是直方图中最多频率对应的那个数或那个范围，而是某个趋势的顶峰，因此可能不止一个，也可能多个（双峰分布），也可能没有，均匀分布；
  2. 中位数：将数据排序，然后选择排序后靠中间的数，该方式适用于正态分布，不太受异常值影响；
  3. 平均数：适用于无异常值或异常值影响不大的情况，不受数据分布情况影响；
  4. 正态分布：整个图形基本符合左右对称；
  5. 正偏斜分布：整个图形的众数靠近左侧；
  6. 负偏斜分布：整个图形的众数靠近右侧；
  7. 不管是哪一种集中趋势，都有其局限性，众数代表的是频率最高的，平均数易受到异常值影响，中位数受分布情况影响；
```
