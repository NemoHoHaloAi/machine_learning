# 分位数理解
```
简单理解：
假设统计了2000个数据（比如对某次考试成绩的统计），
并且从高到低排序，
甲78分，排名1204，1204/2000＝60.2%
乙87分，排名69，69/2000＝3.45%
我们称78分在60.2%的位置上, 87分在3.45%的位置上, 
即在60%的位置上约78分, 3.5%的位置上约87分, 
这就是百分位数的意义.

四分位数的意义是: 分别在25%, 50%, 75%位置上的数.
仍以这2000个数为例, 从高到低排序, 第500个,第100个,第1500个数就是这个总体的四分位数.

python计算方式（使用numpy库）
array_a = [2,3,6,8,10]
```
