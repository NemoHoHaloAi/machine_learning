# 总体与部分
```
情况1. 部分看着都好好的，总体却不行了？？？
  1. 分组A：10(8,2)--80%，90(45,45)--50%
  2. 分组B：50(35,15)--70%，50(20,30)--40%
  咋一看，分组A对应的方式的两个小组的比例都高于分组B的两个小组，但是假设我们合并后呢：
  分组A：8+45,2+45--53%
  分组B：35+20,15+30--55%
  分组A部分看着都要好于分组B，但是总体却不如分组B；
情况2. 总体看着还不错，部分确有问题啊？？？
  与上述相反过来看，分组B总体要好于分组A，但是拆分为部分后却均不如分组A；
本质：由于分组不均导致在部分合并为总体时，各部分产生的影响不均匀，上述例子中，由于每个分组的小组划分不均匀，
  导致在合并为总体时的概率计算中，总体概率会偏向样本量大的那一边，而并不是小组概率的均值，所以会导致这种问题；
```
